Examples of non-breaking spaces:

Initials: A.~S.~Pushkin wrote the novel.

Units of measurement: 10~kg, 5~cm, 20~\% (it is also better to use a non-breaking space with the percent sign).

Reference to a figure: see Fig.~1.

In a sentence, a non-breaking space can be placed before a dash: This is ---~an example.

Note that the line does not break at the position of a non-breaking space.
